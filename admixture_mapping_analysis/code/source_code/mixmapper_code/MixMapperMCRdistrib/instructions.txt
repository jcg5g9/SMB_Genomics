***** DISTRIBUTION FILES *****

- files generated by MCR:
  - run_MixMapper_wrapper.sh
  - MixMapper_wrapper
  - readme.txt
- cmd.txt (edit/replace with the commands you want to run)
- example/ data to check install


***** INSTALLING MCR *****

http://www.mathworks.com/products/compiler/mcr/

- mkdir MCR; cd MCR (unzipping will dump files into current dir)
- wget //www.mathworks.com/supportfiles/MCR_Runtime/R2012a/MCR_R2012a_glnxa64_installer.zip
- uncomment lines of installer_input.txt; save to installer_input_mod.txt:
  - destinationFolder=[INSTALL_PATH]
  - agreeToLicense=yes
  - mode=silent
- ./install -inputFile installer_input_mod.txt


***** RUNNING *****

./run_MixMapper_wrapper.sh [INSTALL_PATH] cmd.txt
